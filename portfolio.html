Personality Test

I coded a personality test using Python. The layout involved the user being asked a series of 38 carefully selected yes or no questions. Then, based on the answers of the user, the test would return 9 different personality and character traits, with the intent to be as accurate as possible. The test would also return a percentage for each of these 9 traits, which would show how strongly each trait can be applied to the user.
